サポートブラウザー・環境¶

Shodoでは以下のブラウザーのみ正式にサポートされています。

Google Chrome最新版
Firefox最新版

以下のOSが正式にサポートされています。 現在、 タブレット、スマートフォン環境は正式にサポートされておりません 。

Windows 10最新版
macOS最新版

推奨されるPC環境は以下になります。 YouTubeやGoogle DriveなどのWebサービスが快適に利用できる程度のスペックが必要となります。

CPUIntel Core i5以上、もしくはApple M1チップ
メモリー8GB以上
インターネット速度5Mbps以上（ダウンロード、アップロード両方）
画面サイズ769px × 769px以上

上記の推奨ではない環境の場合、Shodoが快適に動作しない可能性があります。

今後、お客様のご要望があればサポート環境を拡充させてまいります。 現在のサポート・推奨環境以外でも動作するこがありますが、すべての動作は正式に保証されておりません。
